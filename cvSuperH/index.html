Présentation

Aladdin, est le héros de l'histoire d'Aladdin. C'est un jeune garçon des quartiers populaires qui dort dans la rue et mange ce qu'il arrive à chiper. Pris d'amour pour la belle Jasmine, il fait tout pour obtenir ses faveurs. Avec son cœur pur comme « un diamant d'innocence » il va s'envoler vers de folles aventures.

Expériences professionnelles

1992 : Acteur chez Disney dans Mille et une nuits.

1994 : Acteur chez Disney dans Le retour de Jafar.

1996 : Acteur chez Disney Aladdin et le Roi des voleurs.

Formations

  • Apprenti conducteur de Tapis volants.
  • Apprenti magicien avec Harry Potter.
  • Apprenti voleur chez Ocean's Eleven.
  • Dresseur animalier, Niveau max.

Centres d'intérêts

Le rêve bleu, Jasmine, les enfants, les tapis, le coloriage, le code génie.
